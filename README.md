A collection of high-performance, browser-based games integrated with **Web3 Authentication**. This suite features industry-standard game logic, premium UI/UX design, and secure wallet-sign-in protocols.

---

## 🎮 Included Games

### 1. ♟️ Pro Chess Web3
A grandmaster-level chess experience featuring:
*   **Engine:** `chess.js` logic for 100% move validation.
*   **Security:** Game board remains locked until a cryptographic signature is provided.
*   **UI:** Glassmorphism dashboard with real-time move history tracking.

### 2. 🎲 Elite Ludo
A mathematically precise recreation of the classic board game:
*   **Fair Play:** Randomized dice rolling algorithm.
*   **Multiplayer Ready:** Turn-based logic for Red, Green, Yellow, and Blue.
*   **Visuals:** Dynamic board generation using CSS Grid.

### 3. 🚀 Nebula-X: Space Colony
A complex strategy and resource management simulator:
*   **Visuals:** Real-time HTML5 Canvas particle engine and space environments.
*   **Economy:** Resource mining (Stardust) and Energy Core management.
*   **Persistence:** State-based gameplay locked behind Web3 authorization.

---

## ✨ Key Features

*   **🛡️ Secure Web3 Sign-In:** Uses MetaMask/WalletConnect to ensure only authenticated users can access game functions.
*   **💎 Premium UI/UX:** Built with Tailwind CSS, featuring backdrop blurs, neon borders, and responsive layouts.
*   **⚡ Zero Dependencies:** Optimized to run directly in the browser via CDN—no `npm install` required.
*   **📱 Fully Responsive:** Adaptive layouts for Desktop, Tablet, and Mobile.
